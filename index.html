What is Jott.ly?

Jott.ly is a powerful collaboration tool that allows professional creative teams to unlock their full potential in an interactive virtual environment. This tool allows teams to connect and work with each other on different sources and have key features that make the collaborative creative process as easy and flawless as it can get.

This tool is still in Beta phase and many more features will be implemented through further analysis of creative teams and feedback of users.
